Let \(K\) be a number field and let \(\mathcal{O}_{K}\) be its ring of integers. \(\Re(s)>1\), we define the Dedekind-zeta function to be \[\begin{align} \zeta_K(s) = \sum_{\substack{ \mathcal{I} \subseteq \mathcal{O}_{K} \\ \mathcal{I} \text{ non-zero ideal} } } \frac{1}{\mathop{\mathrm{N}}(\mathcal{I})^{s}}, \end{align}\] where \(\mathop{\mathrm{N}}(\mathcal{I}) = \# \left( \mathcal{O}_{K} / \mathcal{I}\right)\).

Euler expansion

For \(\Re(s) > 1\), we have the following product expansion.

\[\begin{align} \zeta_K(s) = \prod_{\substack{ \mathcal{P} \subseteq \mathcal{O}_{K} \\ \mathcal{P} \text{ prime ideal} } } \frac{1}{\left( 1 - \frac{1}{\mathop{\mathrm{N}}(\mathcal{P})^{s}} \right)}. \end{align}\]

For cyclotomic field

When \(K=\mathbb{Q}(\mu_{n})\), the \(n\)th cyclotomic field, then we have \[\begin{align} \zeta_K(s) = \prod_{ p \in \mathbb{P } } \frac{1}{\left( 1 - \frac{1}{p^{ s\ \mathop{\mathrm{ord}}_{n_{p}}\ p }} \right)^{\frac{\varphi(n)}{\mathop{\mathrm{ord}}_{n_{p}} \ p}} } , \end{align}\] where \(n_{p} = n p^{-\nu_{p}(n)}\), that is \(n\) with all powers of \(p\) removed.

Since \(\mathcal{O}_{K} = \mathbb{Z}[\mu_{n}]\), we can use Dedekind’s factorization lemma. Let \(\Phi_{n}(X) \in \mathbb{Z}[X]\) be the \(n\)th cyclotomic polynomial. So \[\begin{align} & \Phi_{n}(X) \equiv f_1(X)^{e_1}f_{2}(X)^{e_2}\ldots f_r(X)^{e_r} \pmod{p},\\ \Rightarrow & p \mathcal{O}_{K} = \mathcal{P}_{1}^{e_1} \mathcal{P}_{2}^{e_2} \ldots \mathcal{P}_{r}^{e_r} \end{align}\] for some \(r>0\) and \(e_1,e_2,\cdots,e_r \ge 1\). Now \(\mathop{\mathrm{N}}(\mathcal{P})= p^{\deg f_i(X)}\). What are the roots of \(\Phi_n(X)\)? They are primitive \(n\)th roots of unity. In the algebraic closure \(\overline{\mathbb{F}}_p\), the Frobenious automorphism acts on these roots of unity…

Code for computing \(\zeta_{K}(s)\) for real \(s\)

The following sage code computes \(\zeta_{K}(2)\) for any given \(\mathbb{Q}(\mu_n)\).

Caution: Not accurate for large \(n\).

MAX = 10000

## Finding largest power of p that divides n
def power_in(n,p):
    k=0
    while n%p**k==0:
        k=k+1
    return k-1

s=2
n=100


# Finding Dedekind zeta of Q(mu_n) at s=2
DZ=1.00000000
for p in prime_range(1,MAX):
  a=power_in(n,p)
  m=n/p**a
  ord_mp = multiplicative_order(mod(p,m))
  DZ/=(1-1.0/p**(s*ord_mp))**(euler_phi(m)/ord_mp)
print n,DZ
