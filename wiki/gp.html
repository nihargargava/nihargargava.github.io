It is a cool software for computing number theoretical stuff. More commonly known as gp. The official website is here.

Some usage tips

Use ? before a function to see what it does. For example

?bnfinit

To see the long documentation, use ?? instead of ?. To see a lot of related documentation, do ???.

The two important pdfs to look up for general help are users.pdf and refcard-nf.pdf. Both can be found online with a simple web search.

To transpose a vector, add a ~ after it. For example [1,2]~ is a column vector. To count the number of elements in a vector vec, try #vec.
