Subgroup zeta function

Contents

  • References

For a finitely generated subgroup \(\Gamma\), we write \[\begin{equation} a_{\Gamma}(n) = \#\{ H \subseteq \Gamma \text{ is a finite index subgroup } \mid [\Gamma:H] = n\}. \end{equation}\]

One then defines a Dirichlet series of this sequence \[\begin{equation} \zeta_{\Gamma}(n) = \sum_{n=1}^{ \infty } \frac{a_{n}}{n^{s}}, \end{equation}\] for \(\Re(s) \gg 1\).

For \(\Gamma = \mathbb{Z}\), you’re defining the Riemann zeta function.

This zeta function sometimes has often an analytic continuation. See [1].
