Hasse’s theorem

Contents

  • Computational aspects
  • References

(Hasse, 1966, [1])

Define \(\pi_{2}(x)\) as \[\pi_{2}(x) = \#\{ p \ | \ 2 < p \le x\text{ is prime and } p|(2^{m}+1)\text{ for some }m \in \mathbb{Z}_{\ge 0}\}.\] This is the same set as \(\{ p \ | \ 2 < p \le x\text{ is prime and } \text{$\mathop{\mathrm{ord}}_p$}2\text{ is even} \}\).

Then, we have that \[\begin{align} \pi_{2}(x) = \frac{17}{24} \frac{x}{\log x} + o\left(\frac{x}{\log x}\right). \end{align}\]

Computational aspects

Here is a sage program to list such primes upto \(N\).

N=10000

for p in prime_range(3,N):
    ord_mp = multiplicative_order(mod(p,2))
    if ord_mp%2==0:
        print p
