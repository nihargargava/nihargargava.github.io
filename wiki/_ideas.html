Lattice existence lower bounds

Machine learning

This is a good project to transition out of academia. We know that since lattice packing problem is essentially a finite dimensional optimization problem, there must exist a way to recover all the known lower bounds using machine learning. The aim of this project is not to prove anything, but provide stronger evidence towards existing conjectural lower bounds and to showcase the usecase of reinforcement learning in the sphere packing problem.

\(p\)-Selmer groups of Ellipic curves

The \(p\)-Selmer groups of Elliptic curves over Q form \(\mathbb{F}_p\) subspaces. Melanie Woods from Harvard and Ye Tian (AMMS) have some results about the distribution of these Selmer groups. Would it be possible to exploit this to create good lattices?

Ajtai’s worst-case to average case

Ajtai’s construction reduces the worst case of a lattice SVP problem to the average case of some other lattice SVP problem in some other dimension. Can this be used to improve lower bounds?

Some old ideas

  1. Is it true that in a finite group, order of each element is faithfully realized in some irreducible representation?
  2. Connect the multiplicity of the Lovasz number for graphs to the multiplicity of Lovasz number for spherical caps and try to work this out for the “Cube in a sphere” project.
  3. Think about Lasserre heirarchy in context to the “cube in a sphere” project. The cube in a sphere project is basically about finding the independence number of a k-uniform hypergraph.
  4. Find a linear programming bound for number field invariants, like the discriminant, class number etc. Refer to Clozel, Bourgain, Kahane.
