Linear Programming Bound for Sphere Packings

(Cohn-Elkies, 2003)

Suppose \(f: \mathbb{R}^{d} \rightarrow \mathbb{R}\) is an admissible function (i.e.Â Poisson summation works for \(f\)) satisfying the following three conditions.

  • \(f(0)=\hat{f}(0) > 0\),
  • \(f(x) \le 0\) for all \(\| x \| \le r\)
  • \(\hat{f}(t) \ge 0\) for all \(t\).

Then the density of sphere packings in \(\mathbb{R}^{d}\) is bounded above by \(B_{d}(r/2)^{d}\), where \(B_{d}\) denotes the volume of the unit ball in \(\mathbb{R}^{d}\).

Furthermore, there is equality for a lattice \(\Lambda\), if and only if

  • \(f(x)=0\) whenever \(x \in \Lambda \setminus \{ 0\}\),
  • \(\hat{f}(t)=0\) for all \(t \in \Lambda^{*} \setminus \{ 0\}\).
